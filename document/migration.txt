migration

    列表:
        php migration

    migrate your database:
        php migration migrations:migrate

    如果要還原 migrate 的動作
        先查看上次做的版本
            php migration migrations:status
        對這個版本做 down 處理
            php migration migrations:execute  _xxxx_20xxxxxx  --down

    使用的工具:
        Doctrine
